## How to use these samples

1. **Copy the Source**: Open any of the `.md` links above to see how we write for that form.
2. **Run it locally**:
   ```bash
   # Example: Render the manuscript
   draft2final manuscript-sample.md --as manuscript
   ```
3. **Compare**: Open the corresponding PDF to see how the engine handles margins, pagination, and typography.
